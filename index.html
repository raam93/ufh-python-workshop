Python for Complete Beginners · No installation needed

Automate Your Boring Stuff

Skip the repetitive clicking. In 90 minutes you'll learn the core building blocks of Python by solving four tasks that are genuinely tedious to do by hand — sorting files, handling PDFs, cleaning spreadsheets, and tracking changes. All in your browser, no coding experience required.

⏱ 90 minutes 💻 Runs in browser — no installs 🔒 Your files stay on your computer 🐍 No prior coding needed

What you'll learn

Each task is a vehicle for teaching a concept. By the end you'll have working code and the vocabulary to keep learning on your own.

  • Concept
    Understand the smallest units of Python — variables, constants, keywords, and operators — and how to combine them to express logic
  • Concept
    Work with Python's core data structures: integers, floats, strings, lists, tuples, dictionaries, and sets
  • Concept
    Write logic and control flow using if/else, for loops, while loops, and try/except for error handling
  • Concept
    Define reusable functions with parameters and return values — and understand when and why to use them
  • Concept
    Build classes that group related data and operations together, and chain methods to create a readable pipeline
  • Practical
    Sort, filter, and rename hundreds of files automatically based on rules you define
  • Practical
    Merge, split, and extract pages from PDFs — free, locally, without uploading anything
  • Practical
    Clean messy spreadsheets, combine multiple Excel files, generate summary reports, and produce charts — all in a few lines
  • Practical
    Compare two versions of a dataset and automatically surface what rows were added, removed, or changed

The four tasks

Every task is boring, slow, and error-prone when done by hand. That's exactly why Python helps.

  • Task 1
    Smart File Organizer Filter files by prefix, suffix, or pattern — then sort them into folders automatically. Change one constant and the rule changes for all 500 files at once.
  • Task 2
    PDF Toolkit Merge a folder of PDFs, extract a page range, or split a document into individual pages. No uploads, no subscriptions — your documents stay private.
  • Task 3
    Data Cleaning & Analysis Fix every common spreadsheet mess automatically: inconsistent capitalisation, trailing spaces, mixed date formats, missing values, duplicates. Then combine dozens of files, summarise by group, and generate charts.
  • Task 4
    Change Tracker Compare last week's dataset with this week's. Instantly see which rows were added, which were removed, and which values changed.
Why not just ask ChatGPT or Claude? You could — for some of these. But your files are on your computer. Uploading sensitive spreadsheets or contracts to an AI or online tool raises real privacy concerns. Running Python locally means your data never leaves your machine.
